## app_suzu_mori

# 概要
- パートナーとの同棲で発生する課題を解決する。毎日の献立に関する面倒なコミュニケーションを失くす。

# 主要機能
- ユーザ管理
  - サインアップ、サインイン
- パートナー招待
  - 招待コード発行
  - 招待リンク
  - パートナー承認
  - 同棲グループ作成
  - グループ脱退
  - パートナー解除
- 献立管理（要具体化）
  - 献立登録
  - カレンダー
  - 必要な材料の整理
  - お気に入り
  - 履歴
- 買い物リスト
- 冷蔵庫管理
- 買い物リストとの差分表示

- 献立提案（有料）
  - 冷蔵庫から提案
